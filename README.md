# Algoritmos - 1º Semestre

Este repositório reúne exercícios e exemplos desenvolvidos durante o 1º semestre da disciplina de Algoritmos, utilizando a linguagem **Python**.

O objetivo do material é registrar a evolução dos estudos em lógica de programação, entrada e saída de dados, estruturas condicionais, estruturas de repetição e listas.

## Conteúdos Abordados

- Entrada e saída de dados
- Operadores aritméticos
- Estruturas condicionais
- Estruturas de repetição com `for`
- Estruturas de repetição com `while`
- Manipulação de listas
- Resolução de problemas matemáticos e lógicos
- Exercícios práticos em Python

## Descrição das Pastas

### Exercícios

Contém exercícios introdutórios de lógica de programação, com foco em cálculos simples, entrada de dados pelo usuário e exibição de resultados.

Exemplos de temas trabalhados:

- Cálculo de área e perímetro
- Conversão de temperatura
- Cálculo de média
- Consumo de combustível
- Operações matemáticas básicas

### Tomada de decisão

Reúne exercícios com uso de estruturas condicionais, como `if`, `elif` e `else`.

Exemplos de temas trabalhados:

- Validação de valores
- Cálculo de equação do 2º grau
- Regras de decisão
- Classificação de dados
- Problemas com múltiplas condições

### Estrutura de repetição for

Contém exercícios que utilizam a estrutura de repetição `for`, aplicada em contagens, validações e processamento repetitivo de dados.

Exemplos de temas trabalhados:

- Laços com quantidade definida
- Contadores e acumuladores
- Classificação de triângulos
- Repetição de entradas
- Processamento de múltiplos valores

### Estrutura de repetição while

Reúne exemplos e exercícios com a estrutura `while`, utilizada quando a repetição depende de uma condição lógica.

Exemplos de temas trabalhados:

- Laços condicionais
- Validação de entrada
- Repetição até condição de parada
- Contadores e acumuladores

### Listas

Contém exercícios sobre criação, preenchimento e análise de listas em Python.

Exemplos de temas trabalhados:

- Inserção de valores em listas
- Busca pelo maior e menor valor
- Percurso de listas
- Operações básicas com coleções de dados

### Je ne parle toujours pas français

Pasta com exercícios complementares de lógica de programação, envolvendo estruturas condicionais, repetição e resolução de problemas práticos.

## Tecnologias Utilizadas

- **Python**
- **Lógica de Programação**
- **Estruturas Condicionais**
- **Estruturas de Repetição**
- **Listas**

## Como Executar os Arquivos

Para executar qualquer exercício, é necessário ter o Python instalado na máquina.

No terminal, acesse a pasta do arquivo desejado e execute:

```bash
python nome_do_arquivo.py
```

Exemplo:

```bash
python Exercicio01.py
```
